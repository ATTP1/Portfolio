Project AI
M.O.C.H.I

2024 - Work in progress

A friendly AI assistant designed to simplify daily life, built from scratch with clean HTML, SCSS styling, and a touch of 3D magic via Spline. It lives inside a sleek Electron desktop app, responding to questions in real time using the OpenAI API. Beyond chat, it remembers past interactions, offers weather insights, and helps freelancers stay organized with its “Blue” mode, a built-in timer and hour-tracking system. Motivational quotes pop in now and then to keep focus and energy high. With a responsive UI, playful animations, and smart features, it feels less like a tool and more like a helpful digital sidekick.

Project AEC
Street'zza

2025

A responsive restaurant website created as part of a front-end AEC course project. Built using HTML, CSS (with Flexbox and Grid), and Git versioning on GitLab, it includes four main pages: a homepage with a menu, a reservation form, a gallery, and an order summary with tables. The layout adapts cleanly across breakpoints using media queries, with attention to detail in spacing, typography, and responsive design. The project focused on structure, semantics, and visual consistency while respecting wireframes. Each update was carefully tracked using Git, highlighting real-world workflow and development habits.

Electron App
ChronoChou

2024

ChronoChou is a playful and practical desktop timer app created as a first dive into building with Electron. Designed using HTML and styled with CSS, it offers a clean, user-friendly way to clock in and out of work sessions. The logic behind the app—built in JavaScript ;lets users start, stop, and track their time, with a full list of previously recorded sessions. Everything runs locally, with no need for third-party time-tracking tools. From the animated timer to the exportable log of time recorded, the app was built from scratch as a complete front-end and desktop experience. ChronoChou makes time management simple—and marks a proud step in real-world app development.

Project AEC
Level Up

2024-2025

This blog project was created as part of a school assignment in collaboration with a classmate. Following a predetermined wireframe, it was built entirely with HTML, CSS, and JavaScript to demonstrate front-end skills within a defined structure. The layout is fully responsive and adapts across devices, while a 3D model was integrated using Spline to add interactive depth. Although the blog content is fictional, the project highlights the ability to blend clean code, responsive web design, and original creative assets in a cohesive and polished presentation.
